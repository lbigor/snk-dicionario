# Como contribuir

1. Abra uma issue descrevendo o bug ou melhoria antes de codar.
2. Faça fork, crie branch `feat/minha-mudanca` ou `fix/meu-bug` a partir de `main`.
3. Rode `./test.sh` localmente e garanta que passa.
4. Abra PR usando o template — preencha todos os itens do checklist.
5. Aguarde revisão de @lbigor (único que faz merge na `main`).
